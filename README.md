# 寶可夢 DCGAN 產生器

[![Demo CountPages alpha](https://j.gifs.com/yrLZYg.gif)](https://youtu.be/u3849qRYG_8)

使用 tensorflow 練習實做 DCGAN，因為好玩就用了寶可夢當Dataset（721隻 1~6代），用了一些預處理像是：翻轉、亮度、對比微調等...  

共經過8000次更新的結果，可以看到圖像還是蠻扭曲的...😆，在想是不是我樣本不多，而且實際上每隻寶可夢差異極大，所以很難產生五官是正常的圖出來。
